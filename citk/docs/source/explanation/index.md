# Explanation

Conceptual material on conditional-independence testing, the survey
taxonomy `citk` follows, and the v0.1.0 API stability contract.
These pages are aimed at readers who want to understand *why* a
particular test is valid under particular assumptions — or who plan
to cite the library in a paper.
