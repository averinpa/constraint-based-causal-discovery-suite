# PCMCI theory

PCMCI (Runge et al., 2019) recovers a time-series CPDAG over a set
of stationary processes by combining a per-target lagged-parent
selection step (PC₁) with a momentary conditional-independence test
(MCI) that controls for autocorrelation.
